# 兒童發展篩檢問卷 - 15-18 個月、18-24 個月、2-3 歲題目提取

## 集合 4: 15-18 個月
適用對象：15 個月以上、未滿 18 個月兒童

### 粗大動作
1. ★能自己扶著傢俱走或牽手走嗎？
2. 可以(扶著傢俱)踩下踏墊西再站起來？
3. 可以爬上沙發或椅墊或樓梯嗎？

### 精細動作
1. ★可以用指指及食指握筆取貼紙或葡萄乾小物品？
2. 可以把物品放在大人手中？
3. 能把物品放入寬口容器裡？

### 認知語言
1. 孩子可以立即看向大人手指的物品？
2. 玩具被蓋起來，會嘗試去找尋？
3. 聽懂簡單指令(如：拍手、遞來)？
4. ★會發出 ba-ba、ma-ma 等聲音？
5. 會說 1-2 個有意義的詞彙？

### 社會發展
1. ★呼喚名字有反應？
2. 可以和大人玩肢體互動(如：High five)？
3. 眼神接觸地看向施測者？
4. 說「翻翻」會以視覺或手勢回應？

### 評估結果量表
- 粗大動作：通過 ≥ 2，滿分 3
- 精細動作：通過 ≥ 2，滿分 3
- 認知語言：通過 ≥ 3，滿分 5
- 社會發展：通過 ≥ 2，滿分 4

## 集合 5: 18-24 個月
適用對象：18 個月以上、未滿 24 個月兒童

### 粗大動作
1. ★能自己走路且不會經常跌倒？
2. 可以走上樓梯(需要扶著)？
3. 能踢球嗎？
4. 可以跑嗎？
5. 能從坐姿站起來嗎？

### 精細動作
1. ★能用指指及食指握筆取貼紙或葡萄乾小物品？
2. 能用湯匙自己吃飯嗎？
3. 能翻書頁嗎？
4. 能堆積木塊嗎？

### 認知語言
1. 玩具被蓋起來，會嘗試去找尋？
2. 能指出身體部位(如：鼻子、眼睛)嗎？
3. ★會說 10-50 個字？
4. 會說簡單的兩字詞彙嗎？
5. 會執行簡單的指令嗎？

### 社會發展
1. ★呼喚名字有反應？
2. 會模仿大人的簡單動作或表情？
3. 會表現出對食物的喜好？
4. 互動時有自光接觸嗎？
5. 會看著養育者或照顧者？

### 評估結果量表
- 粗大動作：通過 ≥ 3，滿分 5
- 精細動作：通過 ≥ 2，滿分 4
- 認知語言：通過 ≥ 3，滿分 5
- 社會發展：通過 ≥ 3，滿分 5
